# De Rosa Home

Un progetto di design e arredamento realizzato da BettyFe.

## Sviluppo

Il progetto è basato su React, Vite, Tailwind CSS e shadcn/ui.

### Requisiti

- Node.js (v18+)
- npm o bun

### Installazione

```bash
npm install
```

### Avvio in locale

```bash
npm run dev
```

### Build per la produzione

```bash
npm run build
```

## Tecnologie utilizzate

- [React](https://reactjs.org/)
- [Vite](https://vitejs.dev/)
- [Tailwind CSS](https://tailwindcss.com/)
- [shadcn/ui](https://ui.shadcn.com/)
- [Lucide React](https://lucide.dev/)
